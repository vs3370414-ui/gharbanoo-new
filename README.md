# Gharbanoo - Home Services Marketplace

Gharbanoo is a modern marketplace for home services, specifically tailored for Jhanjharpur, Bihar, India. It connects homeowners with reliable experts ranging from plumbers and electricians to house plan designers.

## Features

- **House Plan Design**: Professional 2D and 3D house plans by experienced civil engineers.
- **Expert Discovery**: Find verified experts in various categories (Plumbing, Electrical, Carpentry, etc.).
- **Join as Expert**: A platform for local service providers to grow their business and reach new customers.
- **Responsive Design**: Built with Tailwind CSS and React for a seamless mobile and desktop experience.
- **Real-time Feedback**: Clean animations using Motion for smooth transitions.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS (v4)
- **State Management**: React Hooks
- **Animations**: Motion
- **Icons**: Lucide React
- **Backend / DB**: Firebase (Firestore & Auth)

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vs3370414-ui/gharbanoo-new.git
   cd gharbanoo-new
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Copy `.env.example` to `.env` and fill in your Gemini API key and Firebase configuration if necessary.

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser at `http://localhost:3000`.

## Deployment

The application is designed to be deployed on platforms like Cloud Run or Vercel.
